# Valera game super-duper quest
This is ruby-on-rails project, where user playing with pauper Valera.
Uses Ruby version 2.5.3

## Dependencies
  * Any PC-like device;
  * Installed ruby 2.5.3;
  * Installed NodeJS.

## Configuration
    bundle install
    rails db:migrate
    rails db:seed

## Run the test suite
    bundle exec rspec

## Run the application
    rails server
  follow link http://localhost:3000
